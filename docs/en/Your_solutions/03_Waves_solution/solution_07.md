# 7. Standing Wave Modes

## Given

Length of the string:

L = 80 cm = 0.80 m

Number of antinodes:

n = 4

---

## Important idea

For a standing wave on a string fixed at both ends:

- each antinode corresponds to half of a wavelength segment
- therefore the total string length contains n half-wavelengths

So the relation is:

L = n · (λ / 2)

---

## Step 1: Write the formula

L = nλ / 2

We need to find λ, so solve for wavelength:

λ = 2L / n

---

## Step 2: Substitute the values

λ = (2 × 0.80) / 4

λ = 1.60 / 4

λ = 0.40 m

---

## Explanation

A standing wave with 4 antinodes means the string is divided into 4 equal vibrating parts.

Each part has length λ/2, so:

L = 4 · (λ/2)

That gives the wavelength directly.

---

## Final Answer

The wavelength is:

λ = 0.40 m
