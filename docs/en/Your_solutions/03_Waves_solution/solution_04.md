# 4. Standing Wave

## Given

A standing wave on a string is described by:

y(x,t) = 2A sin(kx) cos(ωt)

We need to:

1. identify the positions of nodes  
2. identify the positions of antinodes  
3. explain the physical meaning

---

## Step 1: Understand the equation

The equation

y(x,t) = 2A sin(kx) cos(ωt)

represents a standing wave.

It is different from a traveling wave because the spatial part and the time part are separated:

- sin(kx) depends only on position
- cos(ωt) depends only on time

This means the shape does not move left or right.  
Instead, different points of the string oscillate up and down with different amplitudes.

---

## Step 2: Find the nodes

Nodes are points that never move.

That means the displacement must always be zero for all times:

y(x,t) = 0

From the equation:

y(x,t) = 2A sin(kx) cos(ωt)

the only factor that can be permanently zero is:

sin(kx) = 0

Now solve:

kx = nπ

where n = 0, 1, 2, 3, ...

So the node positions are:

x = nπ / k

---

## Step 3: Find the antinodes

Antinodes are points where the oscillation amplitude is maximum.

In the standing wave equation, the amplitude at position x is:

Amplitude = 2A |sin(kx)|

This is maximum when:

|sin(kx)| = 1

That happens when:

kx = π/2 + nπ

So:

x = (π/2 + nπ) / k

or equivalently:

x = (2n + 1)π / 2k

where n = 0, 1, 2, 3, ...

These are the antinode positions.

---

## Step 4: Physical interpretation

- At the nodes, the string never moves.
- At the antinodes, the string moves with the largest amplitude.
- The standing wave is formed by the superposition of two identical waves traveling in opposite directions.

This means energy does not propagate along the string in the usual way.  
Instead, the string vibrates in a fixed pattern.

---

## Final Answer

Node positions:

x = nπ / k,   n = 0, 1, 2, 3, ...

Antinode positions:

x = (2n + 1)π / 2k,   n = 0, 1, 2, 3, ...

The wave is a standing wave with fixed nodes and oscillating antinodes.
