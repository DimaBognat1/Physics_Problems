# 3. Superposition Principle

## Given

Two waves travel in the same medium:

y₁(x,t) = A sin(kx − ωt)

y₂(x,t) = A sin(kx − ωt + π)

We need to find the resulting wave.

---

## Step 1: Compare the two waves

The second wave differs from the first one by a phase shift of π.

We use the identity:

sin(α + π) = −sin(α)

So:

y₂(x,t) = −A sin(kx − ωt)

---

## Step 2: Add the two waves

The total displacement is:

y(x,t) = y₁(x,t) + y₂(x,t)

Substitute both expressions:

y(x,t) = A sin(kx − ωt) + (−A sin(kx − ωt))

y(x,t) = 0

---

## Explanation

The two waves have the same amplitude, the same frequency, and the same wavelength, but they are shifted by π.

This means they are in opposite phase, so they cancel each other completely.

This is called destructive interference.

---

## Final Answer

The resulting wave is:

y(x,t) = 0

The waves completely cancel each other.
