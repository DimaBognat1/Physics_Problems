# Section 1 — Kinematics

We are given:

r(t) = (a cos(ωt), b sin(ωt), bt)

---

## Solution

### a) Trajectory

From x and y:

x = a cos(ωt)  
y = b sin(ωt)  

Eliminate t:

(x/a)² + (y/b)² = 1  

This is an ellipse.

z = bt → linear growth  

So trajectory is a helix (spiral).

---

### b) Path length

Velocity:

v(t) = (−aω sin(ωt), bω cos(ωt), b)

Speed:

|v| = √[(aω sin(ωt))² + (bω cos(ωt))² + b²]

Path length:

L = ∫ |v(t)| dt from 0 to t₀  

---

### c) Interpretation

- If a = b → circular helix  
- If b = 0 → planar ellipse  
- If ω = 0 → straight line  

---

## Final Answer

Trajectory: elliptical helix  

Path length:

L = ∫ √[(aω sin(ωt))² + (bω cos(ωt))² + b²] dt  

Special cases described above.
