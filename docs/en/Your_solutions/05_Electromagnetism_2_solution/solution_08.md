# 8. EM Spectrum

## Given

We need to list the following types of electromagnetic radiation in order of increasing wavelength:

- Infrared
- Ultraviolet
- Microwaves
- X-rays
- Radio waves
- Gamma rays

---

## Step 1: Understand the electromagnetic spectrum

In the electromagnetic spectrum, radiation types are ordered by wavelength.

From shortest wavelength to longest wavelength, the order is:

gamma rays → X-rays → ultraviolet → visible light → infrared → microwaves → radio waves

---

## Step 2: Select only the given radiation types

The problem gives:

Gamma rays, X-rays, Ultraviolet, Infrared, Microwaves, Radio waves

So we arrange them from the shortest wavelength to the longest wavelength.

---

## Step 3: Write the order of increasing wavelength

Increasing wavelength means going from the smallest wavelength to the largest wavelength.

Therefore:

Gamma rays < X-rays < Ultraviolet < Infrared < Microwaves < Radio waves

---

## Explanation

Gamma rays have the shortest wavelengths and the highest frequencies.

Radio waves have the longest wavelengths and the lowest frequencies.

As wavelength increases, frequency decreases because:

c = λf

---

## Final Answer

The order of increasing wavelength is:

Gamma rays → X-rays → Ultraviolet → Infrared → Microwaves → Radio waves
