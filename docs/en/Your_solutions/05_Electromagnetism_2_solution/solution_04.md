# 3. Biot-Savart Law

## Given

A small segment of a straight wire has length:

L = 0.1 m

The current in the segment is:

I = 3 A

The point P is located at a distance:

r = 0.2 m

The current segment is perpendicular to the line connecting it to point P.

We need to calculate the magnetic field at point P.

---

## Step 1: Use the Biot-Savart Law

For a small current element, the magnetic field is given by:

B = μ₀ / (4π) · I L sinθ / r²

where:

- B is the magnetic field
- μ₀ = 4π × 10^(-7) T·m/A
- I is the current
- L is the length of the current segment
- r is the distance to the point
- θ is the angle between the current segment and the line to point P

---

## Step 2: Determine the angle

The problem says that the segment is perpendicular to the line connecting it to point P.

Therefore:

θ = 90°

and:

sin90° = 1

So the formula becomes:

B = μ₀ / (4π) · I L / r²

---

## Step 3: Substitute the values

We know:

μ₀ / (4π) = 10^(-7)

I = 3 A

L = 0.1 m

r = 0.2 m

So:

B = 10^(-7) · (3 · 0.1) / (0.2)²

---

## Step 4: Calculate

First calculate the numerator:

3 · 0.1 = 0.3

Now calculate the denominator:

(0.2)² = 0.04

So:

B = 10^(-7) · 0.3 / 0.04

B = 10^(-7) · 7.5

B = 7.5 × 10^(-7) T

---

## Explanation

The Biot-Savart Law tells us how much magnetic field is created by a small part of a current-carrying wire.

The field is strongest when the current element is perpendicular to the line connecting it to the observation point.

Here the angle is 90°, so sinθ = 1, and the magnetic field has its maximum value for this geometry.

---

## Final Answer

The magnetic field at point P is:

B = 7.5 × 10^(-7) T
