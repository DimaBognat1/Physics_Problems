# Section 0 — Function Analysis

We are given the function:

f(x) = 3x² − 12x + 7  

We need to find local maxima or minima.

---

## Solution

### Step 1: Compute derivative

f'(x) = 6x − 12  

---

### Step 2: Find critical point

Set derivative to zero:

6x − 12 = 0  

6x = 12  

x = 2  

---

### Step 3: Determine type (second derivative)

f''(x) = 6  

Since f''(x) > 0 → this is a minimum

---

### Step 4: Find function value

f(2) = 3(2²) − 12(2) + 7  

= 12 − 24 + 7  

= −5  

---

## Final Answer

The function has a minimum at:

x = 2  
f(x) = −5
