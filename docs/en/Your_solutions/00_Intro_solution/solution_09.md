# Section 0 — Optimization Problem

We are given:

y = 3 − x²  

We need to find the rectangle of maximum area under the curve.

---

## Solution

### Step 1: Area function

A = x * y = x(3 − x²)  

A(x) = 3x − x³  

---

### Step 2: Derivative

A'(x) = 3 − 3x²  

Set to zero:

3 − 3x² = 0  

x² = 1  

x = 1 (first quadrant)

---

### Step 3: Find height

y = 3 − 1² = 2  

---

## Final Answer

Maximum area rectangle:

width = 1  
height = 2
