# Section 0 — Trigonometry

We are given a vector A with:

|A| = 15  
θ = 60°  

We need to find its horizontal and vertical components.

---

## Solution

### Step 1: Use formulas

Ax = A cosθ  
Ay = A sinθ  

---

### Step 2: Substitute values

cos60° = 1/2  
sin60° = √3 / 2  

---

### Step 3: Calculate components

Ax = 15 * (1/2) = 7.5  

Ay = 15 * (√3 / 2) = (15√3) / 2  

---

## Final Answer

Ax = 7.5  
Ay = (15√3) / 2
